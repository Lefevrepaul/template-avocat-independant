# Image de fond Lyon

Pour que les sections hero affichent correctement l'image de Lyon en arrière-plan, veuillez ajouter une image nommée :

**`lyon-background.jpg`**

dans ce dossier (`public/images/`).

## Recommandations pour l'image :

- **Format** : JPG ou PNG
- **Résolution** : Minimum 1920x1080px (Full HD) pour une bonne qualité
- **Aspect** : Paysage (format 16:9 recommandé)
- **Contenu** : Vue panoramique de Lyon (par exemple : vue depuis Fourvière, quai du Rhône, Presqu'île, etc.)
- **Style** : Photo professionnelle, lumineuse, avec un bon contraste

## Exemples de sujets pour l'image :

- Vue panoramique depuis la colline de Fourvière
- Quai du Rhône avec les bâtiments historiques
- Presqu'île de Lyon avec ses rues piétonnes
- Place Bellecour
- Vue aérienne de Lyon

L'image sera automatiquement utilisée comme arrière-plan sur toutes les pages hero avec un overlay blanc semi-transparent pour assurer la lisibilité du texte.
