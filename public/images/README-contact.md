# Image de fond pour les sections de contact

Pour que les sections de contact en fin de page affichent correctement l'image de fond, veuillez ajouter une image nommée :

**`contact-background.jpg`**

dans ce dossier (`public/images/`).

## Recommandations pour l'image :

- **Format** : JPG ou PNG
- **Résolution** : Minimum 1920x1080px (Full HD) pour une bonne qualité
- **Aspect** : Paysage (format 16:9 recommandé)
- **Contenu** : Image professionnelle et sobre (bureau, architecture, ou autre image en rapport avec le domaine juridique)
- **Style** : Photo professionnelle, avec un bon contraste

## Exemples de sujets pour l'image :

- Bureau professionnel
- Architecture moderne
- Vue intérieure sobre
- Éléments juridiques (livres, documents)
- Ambiance professionnelle et rassurante

L'image sera automatiquement utilisée comme arrière-plan sur toutes les sections de contact en fin de page avec un overlay bleu semi-transparent (80% d'opacité) pour assurer la lisibilité du texte blanc.
